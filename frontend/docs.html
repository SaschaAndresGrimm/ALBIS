ALBIS — ALBIS WEB VIEW

ALBIS is an ALBULA‑style, browser‑based viewer for diffraction images and large HDF5 stacks. It is platform‑independent, free, and open source.

It targets modern DECTRIS detectors (SELUN, EIGER2, PILATUS4) and supports filewriter1 and filewriter2 layouts, including multi‑threshold data.

Project note: this is a private vibe‑coding project for fun and educational purposes.

Core Features

  • File → Open loads HDF5 stacks and common detector formats (.tif/.tiff, .cbf/.cbf.gz, .edf).
  • Top toolbar: compact frame slider, frame index, step dropdown, FPS dropdown, and playback controls.
  • Data tab: file source selection, file inspector/header viewer, and Series Operations (sum/mean/median).
  • Multi‑threshold selector for filewriter2 datasets.
  • Data sources: File, Watch Folder, SIMPLON Monitor (live TIFF stream), and Remote Stream.
  • Pixel mask support with gap/defect highlighting.
  • Histogram with draggable BG/FG and inline Auto Contrast / Auto controls.
  • Overlay tab: Statistics and ROI, Resolution Rings, Peak Finder.
  • ROI statistics default to full-image values when ROI mode is set to None.
  • Settings dialog (Settings menu) edits albis.config.json with Save / Save & Close.
  • WebGL2 rendering with CPU fallback.

Remote Stream

  • Use Data Source = Remote Stream and set a source id (default: default).
  • Send frames via POST /api/remote/v1/frame using multipart upload.
  • The sender source_id must match the UI source id.
  • 204 No Content from /api/remote/v1/latest means “no new frame yet”.
  • For raw frame bytes, provide meta JSON with format=raw, dtype, and shape.
  • Optional metadata updates ring parameters (distance, pixel size, energy, beam center).
  • Optional peak_sets render colored external peak overlays in the image.

Tips

  • Mouse Wheel zoom at cursor.
  • Double‑Click zoom in.
  • Right‑Drag define ROI.
  • Arrow Keys frame or threshold navigation.
