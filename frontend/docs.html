ALBIS — ALBIS WEB VIEW

ALBIS is a local, ALBULA‑style HDF5 viewer for large stacks. It focuses on fast navigation, robust contrast controls, and inspection tools for diffraction data.

Core Features

  • File → Open uploads `.h5/.hdf5` stacks using the system file picker (or opens in place).
  • Dataset selection, frame slider, step, play/pause and FPS controls.
  • Auto Load: watch folder for latest `.h5/.hdf5`, `.tif/.tiff`, or `.cbf` file (filterable + glob pattern).
  • Browse… opens a native folder picker (macOS) to select absolute paths.
  • Auto Load: SIMPLON monitor polling for live TIFF frames (LIVE badge) with pixel mask prefetch.
  • Fit‑to‑window, cursor‑centered zoom, double‑click zoom, and drag to pan.
  • Overview panel with viewport rectangle and drag‑resize zoom.
  • Histogram with log X/Y, auto‑contrast (log‑percentiles), and draggable BG/FG.
  • ROI tools: line, box, circle, annulus with stats and plots.
  • Optional detector pixel mask (gaps at 0, defective pixels in blue).
  • Heat (default), HDR, Grey, Viridis, Magma, Inferno, Cividis, Turbo color maps.

Tips

  • Mouse Wheel zoom at cursor.
  • Double‑Click zoom in (cursor‑centered).
  • Right‑Drag define ROI.
  • Alt/Option while resizing overview keeps center fixed.
  • Pixel labels appear at high zoom and can be toggled in the Zoom section.
