ALBIS — ALBIS WEB VIEW

ALBIS is an ALBULA‑style, browser‑based viewer for diffraction images and large HDF5 stacks. It is platform‑independent, free, and open source.

It targets modern DECTRIS detectors (SELUN, EIGER2, PILATUS4) and supports filewriter1 and filewriter2 layouts, including multi‑threshold data.

Project note: this is a private vibe‑coding project for fun and educational purposes.

Core Features

  • File → Open loads HDF5 stacks and common detector formats.
  • Dataset inspector with frame slider, step, and playback.
  • Multi‑threshold selector for filewriter2 datasets.
  • Data sources: File, Watch Folder, SIMPLON Monitor (live TIFF stream).
  • Pixel mask support with gap/defect highlighting.
  • Histogram with draggable BG/FG and auto‑contrast.
  • ROI tools with statistics and projections.
  • WebGL2 rendering with CPU fallback.

Tips

  • Mouse Wheel zoom at cursor.
  • Double‑Click zoom in.
  • Right‑Drag define ROI.
  • Arrow Keys frame or threshold navigation.
