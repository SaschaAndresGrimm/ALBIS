ALBIS is a canvas-first viewer for diffraction images and large HDF5 stacks. It is lightweight, cross-platform, and open source.

It supports common detector workflows (file and live sources), ROI and overlay analysis, and frame-by-frame navigation with high rendering performance.

New users: start with File -> Open and then use the right panel tabs (View, Data, Overlay) for analysis controls.

Quick Start

  • Open data: File -> Open to load HDF5, TIFF, CBF(.gz), or EDF files.
  • Navigate frames with slider, index field, play/pause, and keyboard arrows.
  • Use the Data tab to select data source, dataset, threshold, and inspect file metadata.
  • Use the Overlay tab for ROI stats, resolution rings, and peak finder.
  • Use Settings -> Preferences to configure server, viewer, data access, and logging.

Interaction Basics

  • Mouse wheel zoom at cursor.
  • Left-drag pan image (also works while zoomed out).
  • Right-drag define ROI (when ROI mode is active).
  • Double-click quick zoom in.
  • F toggle full screen.
  • F1 open this documentation page.

Data Sources and Streaming

  • File: load local files from disk.
  • Watch Folder: poll folders for new files.
  • SIMPLON Monitor: monitor live TIFF updates.
  • Remote Stream: consume pushed frames via API source ids.
  • For technical API details, see the repository README.

Troubleshooting

  • If the viewer shows OFFLINE, wait for backend startup or restart the launcher.
  • If no image appears, check selected dataset and threshold in the Data tab.
  • If overlays look stale in playback, pause and re-enable the overlay tool once.
  • Use Help -> Open Log File for runtime diagnostics.
