# Security Policy

If you discover a security issue, please report it responsibly.

## Reporting

Open a GitHub issue with a **[security]** prefix, or contact the maintainers directly
if the issue includes sensitive information.

## Scope

ALBIS is typically run on local machines and beamline networks. Please highlight:
- Any data exposure issues
- Remote access or authentication bypass risks
- Unsafe default configurations

We will respond as soon as practical.
